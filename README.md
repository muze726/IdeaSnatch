# 灵感捕手（Chrome / Edge 扩展）

你在网页上看到好标题、好表达、好选题时，点一下浏览器右上角图标，就能把当前页面的**标题 / 链接 / 选中文本**保存下来，并补充**备注 / 标签**。数据保存在浏览器本地，可导出给 AI 做后续分析。

## 功能

- 自动抓取：当前网页标题、网页链接、你刚选中的文字
- 手动补充：备注、标签（**回车添加多个标签**，也支持逗号分隔/粘贴）
- 一键保存：保存在浏览器本地
- 重复链接提醒：同一个链接已保存过会提示是否继续保存
- 删除记录：可在“最近保存”列表中删除
- 最近记录：弹窗内展示**最新 10 条**，并显示**总保存条数**
- 捕捉区可折叠：标题/链接/选中文本默认折叠成摘要，减少弹窗高度
- 一键导出：导出全部灵感为 **CSV / JSON**（JSONL 仍保留但作为次要选项）

## 怎么安装到 Chrome / Edge 里试用

### 1) 准备

本项目的扩展源码在 `extension/` 目录。

### 2) Chrome 安装（开发者模式）

1. 打开 Chrome，进入扩展管理页：`chrome://extensions/`
2. 右上角打开“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本项目的 `extension/` 文件夹
5. 现在浏览器右上角会出现“灵感捕手”图标，点它即可使用

### 3) Edge 安装（开发者模式）

1. 打开 Edge，进入扩展管理页：`edge://extensions/`
2. 打开“开发人员模式”
3. 点击“加载解压缩的扩展”
4. 选择本项目的 `extension/` 文件夹

## 使用方法（最短路径）

1. 在任意网页上，用鼠标选中一段文字（可选）
2. 点击右上角“灵感捕手”图标，弹窗会自动填入标题 / 链接 / 选中文本
3. “捕捉信息”默认是折叠的：需要看详情时点一下即可展开
4. 填写备注与标签（可选）：标签框输入一个词后按回车可继续加下一个
5. 点击“保存灵感”
6. 下方“最近保存（最新 10 条 / 共 X 条）”会出现刚保存的记录；点击记录可打开原链接；右侧 × 可删除
7. 点击“导出 CSV / 导出 JSON”即可把全部数据导出为文件（交给 AI 做选题分析）

## 导出文件说明

三种导出都包含同一套字段，只是格式不同：

- **JSON**：一个数组，适合直接丢进很多工具/脚本里
- **CSV**：表格格式，方便在 Excel / 表格工具里查看，也方便很多 AI 工具读取
- **JSONL**：一行一个 JSON 对象，适合给数据管道做流式处理（当前界面里作为次要按钮）

字段（每条记录）：

- `id`：记录唯一 id
- `createdAt`：保存时间（ISO 字符串）
- `title`：页面标题
- `url`：页面链接
- `selectionText`：你当时选中的文字
- `note`：备注
- `tags`：标签数组

## 测试清单（建议按顺序）

### A. 自动抓取是否正确

- 打开任意网页，**不选中文字** → 点图标 → 弹窗里标题/链接应正确，选中文本为空
- 在网页里选中一段文字 → 点图标 → 弹窗里“选中文本”应显示你选的内容
- 打开一个新网页重复上述动作 → 标题/链接应跟着变化

### B. 捕捉区折叠是否可用

- 打开弹窗 → 默认应看到“捕捉信息”的一行摘要（折叠状态）
- 点一下“捕捉信息” → 应展开显示标题/链接/选中文本
- 再点一下 → 应收起；关闭弹窗再打开 → 折叠状态应保持

### B. 保存是否生效

- 备注填一段话、标签依次输入 `标题参考` 回车、`金句` 回车 → 点“保存灵感”
- 下方“最近保存”应出现这条记录
- 关闭弹窗再打开 → “最近保存”仍应存在（证明数据已落地）

### C. 最近 10 条展示 + 总数

- 连续保存 11 条 → 列表只显示 10 条；标题处的“共 X 条”应显示 11
- 点击任意一条最近记录 → 应在新标签页打开对应链接
- 点击右侧 × 删除其中一条 → 列表更新，总数减少

### D. 导出是否可用

- 点“导出 CSV” → 保存后用 Excel 打开，中文应正常显示，列应不乱
- 点“导出 JSON” → 保存后用文本编辑器打开，检查是否包含你刚刚保存的记录内容（标题/链接/备注/标签）

## 常见问题

### 1) 为什么有时拿不到选中文本？

少数页面（例如浏览器内置页面、扩展商店、某些受限制页面）不允许扩展读取页面内容；这时标题和链接通常还能显示，但选中文本会拿不到。
